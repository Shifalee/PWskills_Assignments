<!--What are HTML Entities? List out 5 commonly used HTML entities?-->

HTML Entities:-HTML entities are pieces of text that start with an ampersand ( & ) and
               end with a semicolon ( ; ). They are used to display reserved characters
               and invisible characters.HTML entities are used to replace reserved characters
               and characters that are not present on a keyboard. For example, the greater than
               ( > ) or lesser than ( < ) symbol. 

5 Commonly used HTML Entities-:
 
1. &lt; - Represents the less-than symbol <.
   Example: <p>This is an example &lt;em&gt;text&lt;/em&gt;.</p>

2. &gt; - Represents the greater-than symbol >.
   Example: <p>For more information, visit our website &gt; <a href="#">here</a>.</p>

3. &amp; - Represents the ampersand symbol &.
   Example: <p>For further details, contact us &amp; we'll assist you.</p>

4. &quot; - Represents a double quotation mark "
   Example: <p>The title of the book is "The Great Gatsby".</p>

5.&copy; - Represents the copyright symbol Â©.
  Example: <p>&copy; 2023 ACME Corporation. All rights reserved.</p>
